# プライバシーポリシー（Privacy Policy）

最終更新日: 2025-12-17

VoiceTagEditor（以下「本アプリ」）は、ユーザーのプライバシーを尊重します。本アプリはデスクトップアプリとしてローカル環境で動作し、原則としてユーザーのデータを外部に送信しません。

## 1. 収集する情報

本アプリは、以下の情報を本アプリ内の処理に必要な範囲で取り扱います。

- ユーザーが選択/ドラッグ&ドロップした音声ファイル・画像ファイル・フォルダのパス
- 音声ファイルに含まれるメタデータ（タイトル、アーティスト、アルバム、カバーアート等）

## 2. 外部への送信（通信）

- 本アプリは、解析・変換・編集したデータを開発者や第三者へ自動送信しません。
- 解析/変換処理のために、OSにインストール済みの FFmpeg を外部プロセスとして呼び出します（FFmpegは本アプリに同梱しません）。

## 3. ローカルへの保存

本アプリは利便性のため、以下をローカルに保存する場合があります。

- オンボーディング（ガイド）の表示済みフラグ: `localStorage`（キー: `vte_onboarding_seen_v1`）
- アルバムアートの一時キャッシュ: 例）`$HOME/.cache/VoiceTagEditor/album_art/`（環境により `USERPROFILE` 配下）

## 4. 削除方法

- オンボーディングの表示状態をリセットしたい場合は、WebView のローカルストレージを削除することで初期化できます。
- アルバムアートのキャッシュを削除したい場合は、上記キャッシュフォルダを削除してください。

## 5. 第三者ソフトウェア

本アプリは Tauri / Preact などのOSSを利用しています。詳細は `THIRD-PARTY-NOTICES.md` を参照してください。

## 6. 問い合わせ

不明点や要望は GitHub リポジトリの Issues へお願いします。
